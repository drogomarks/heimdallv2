Getting Started


This page is designed to help you get started using the apiTool.py.

So we'll go over the things you need to be able to run this without any issues on your local machine. Again, this is for the apiTool.py Python script, the web tool works as is.


Requirements:

  • Your API User Key,Secret Key and User Agent
  • Python 2.7
  • The apiTool.py script

Set Up


  Windows

  1. Install Python on your machine. Remember to use 2.7!

  2. Download the script. NOTE the location you download it too.

  3. Edit the file with whatever editor you wanat and look for lines 67-69. You need to update the usrkey,scrtkey and usr_agent variables with the correct information which you can get from your team lead or boss.

  4. Press Windows Key + Type in "cmd" + press ENTER. This should pull up command prompt.

  5. Navigate to the location of the script. For example if it's on your desktop:

    6.  > C: (press ETNER)

    This moves you to the C: drive, this may or may not be where need to go, double check.

     C:\> cd Users\myusername\Desktop (press ETNER)

    You should be in whatever directory you downloaded the script to. You can check with the "dir" command.

     python apiTool.py -h (press ENTER)

    You should get something like this:

     C:\Users\myusername\Desktop>python apiTool.py -h (press ENTER)
      usage: apiTool.py [-h] -c CUSTOMER_ID -d DOMAIN [-t [{ex,rs}]] [-u USER]
                        [--get-user-info] [--get-domain-info] [--get-mailboxes]
                        [--user-vex {false,true}] [--domain-vex {false,true}]

      optional arguments:
        -h, --help            show this help message and exit
        -c CUSTOMER_ID, --customer-id CUSTOMER_ID
                              Email & Apps Customer ID
        -d DOMAIN, --domain DOMAIN
                              Customer domain name you are working with ex:
                              domain.com.
        -t [{ex,rs}], --mbx-type [{ex,rs}]
                              Mailbox Type: rs or ex
        -u USER, --user USER  User/mailbox you want to work with.
        --get-user-info       User/mailbox you want to GET info on.
        --get-domain-info     Domain you want to GET info on.
        --get-mailboxes       Get list of mailbox users for a domain.
        --user-vex {false,true}
                              User/Mailbox you want to change value for.
        --domain-vex {false,true}
                              domain you want to change value for.

  Mac

  1. Lucky for you Python comes ready to roll on Mac so no need to install it! :D

  2. Download the script. NOTE the location you download it too.

  3. Edit the file with whatever editor you wanat and look for lines 67-69. You need to update the usrkey,scrtkey and usr_agent variables with the correct information which you can get from your team lead or boss.

  4. Open Terminal: Finder > Applications > Terminal

  5. Navigate to the location where downloaded the script:

    6. your-macbook:~ username$ cd /path/to/where/script/is (press ENTER)
    your-macbook:~ username$ python apiTool.py -h
