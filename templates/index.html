Welcome to Heimdallv2!


What is Heimdall?

Heimdall is a web tool that interacts with the Rackspace Email and Apps API on your behalf. It allows you to fill out a web form with specific parameters to either get information or make a change to mailbox user, domain etc.


You can read more about the API and what you can do with it here.


What's New in v2?

There were some issues with the PHP version and it was decided Python would be much better at this, so it was rebuilt in a Python Framework.

We had issues previously with requests against domains with large amounts of users, this should no longer be an issue.

There is now a Python Script you can use locally, that takes arguments to perform all these same tasks if you prefer to do it that way. There is a Getting Started page that talks about stuff you need to use it, and a Examples page so you easily copy/paste commands with it and just updated the arguments to suite your needs.

The goal here was to make these tasks as simple as possible for you, so please provide feedback if it's failing at that in any way, or how it can be better. You can use the Feeback page to send suggestions or things you need it to do.


Current Features:

  • GET Details about a mailbox user from the API.
  • GET Details about a domain from the API.
  • GET a list of users in a domain from the API.
  • PUT call to change the value of visibleInExchangeGAL to true or false for a single user
  • PUT call to change the value of visibleInExchangeGAL to true or false for all users in a domain.
